# Stitch Contact PillarVolt

Overview of the Stitch Contact PillarVolt project. This project is a Next.js application designed for PillarVolt, featuring a modern, responsive design with a focus on showcased services, projects, and news.

## Features

-   **Next.js 15+**: Built with the latest Next.js features including App Router.
-   **TypeScript**: Fully typed for better code quality and developer experience.
-   **Tailwind CSS**: Utility-first CSS framework for rapid and responsive UI development.
-   **Responsive Design**: Optimized for all device sizes, from mobile to desktop.
-   **SEO Optimized**: configured with proper metadata and semantic HTML.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To create an optimized production build:

```bash
npm run build
```

The output will be in the `.next` folder.

## Deployment

This project is ready to be deployed on Vercel or any other Next.js compatible hosting provider.

Check out [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
